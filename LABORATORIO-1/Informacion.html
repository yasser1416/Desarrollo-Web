TECNOLOGIAS Y DESARROLLO WEB (SIS-256)

1. DESCRIPCIÓN Y JUSTIFICACIÓN DE LA ASIGNATURA

Internet se han convertido en un producto estratégico de primer orden, todas las empresas modernas han empezado a desarrollar sus sitios web y a construir en sus empresas Intranets y Extranets, llegando a construir sitios completos mediante aplicaciones que funcionan en estos entornos.
En un mundo tan comunicado como el actual el estudiante de la carrera de ingeniería de sistemas debe estar preparado para enfrentar y ser capaz de planificar, diseñar, desarrollar, implementar y mantener aplicaciones que funciona en la web. La asignatura de Tecnologias y Desarrollo Web permitirá al estudiante introducirse en el mundo de las tecnologias de internet y el desarrollo de aplicaciones Web, comprendiendo elfuncionamineto de internet y el web, las características que tiene este tipo de aplicaciones, para desarrollar habilidades prácticas en este temática.

2. RELACIONES DE LA ASIGNATURA

La asignatura de Tecnologia y Desarrollo Web esta directamente relacionado con todo el área de desarrollo, es decir que el estudiante debe tener conocimiento de programación, también el manejo de Base de Datos, formalmente en el plan de estudios el requisito para cursarla es SIS 304 Base de Datos II y SIS 407 - Interaccion Humano Computador

3. OBJETIVO GENERAL

El alumno comprenderá los fundamentos para el desarrollo de aplicaciones en entornos orientadas al Web de una manera efectiva, conocer las tecnologías asociadas, estará capacitado para monter servidores web, desarrollar aplicaciones que funcionen en internet con acceso a base de datos y tecnologia ajax que mejore la experiencia del usuario .

4. OBJETIVOS ESPECÍFICOS

Conocer el funcionamiento de internet y la Web
Ser capaz de intalar y configurar un servidor web
Ser capaz de utilizar los lenguajes de marcas XML, HTML
Separar la presentación del contenido en páginas web con el uso de CSS
Ser capaz de desarrollar aplicaciones Web de lado del servidor con el lenguaje de programación PHP.
Disponer de una visión de las tecnologías del lado de cliente y ser estar preparado para desarrollar script de cliente web utilizando Javascript.
Comprender el funcionamiento de Tecnologías RIA, SOFEA y estar preparado para desarrollar aplicaciones web utilizando las tecnologías
Escribir aplicaciones Seguras y manejo dela seguridad en internet
